# Audio Files

These are audio files in the following formats:
| Extension | Full Name        |
|-----------|------------------|
| .ogg      | Ogg Vorbis Audio |
| .mp3      | MP3 Audio        |
| .m4a      | MPEG-4 Audio     |

## Naming Convention

File code system is derived from Johnny.Decimal.

First number is the topic,
the following three numbers are the ID,
Then a dash (-) and the name with no spaces.

Example: ``5001-aah.ogg``  
Topic is Other, ID is 001, name is "aah"

### Topics key:
| Number | Topic              |
|--------|--------------------|
| 0      | General            |
| 1      | Dungeons & Dragons |
| 2      | Film               |
| 3      | Music              |
| 4      | Reading            |
