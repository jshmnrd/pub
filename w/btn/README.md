# 88x31 Button Files

This folder contains more than one file type, but they are all 88x31 buttons.
| Extension    | Full Name                          |
|--------------|------------------------------------|
| .jpg / .jpeg | JPEG Image (creative, I know)      |
| .png         | Portable Network Graphic           |
| .gif         | Graphical Interchange Format File  |
| .webp        | WebP Image (again, creative...).   |
| .svg         | Scalable Vector Graphic            |

## Naming Convention

This folder's naming convention deviates slightly from the other folders in /w/. It doesn't use a topic, just an ID, this time 4-digits instead of the usual 3.

The first four digits are the ID,
Then a dash (-) and the name with no spaces.

Example: ``0002-jshmnrd-10.jpg``  
ID is 0002, name is "jshmnrd-10"
