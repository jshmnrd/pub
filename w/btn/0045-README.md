# Image Files

These are image files in a raster format:
| Extension    | Full Name                          |
|--------------|------------------------------------|
| .jpg / .jpeg | JPEG Image (creative, I know)      |
| .png         | Portable Network Graphic           |
| .gif         | Graphical Interchange Format File  |
| .webp        | WebP Image (again, creative...).   |

There's more than this, I just will only ever use these on here.

## Naming Convention

File code system is derived from Johnny.Decimal.

First number is the topic,
the following three numbers are the ID,
Then a dash (-) and the name with no spaces.

Example: ``1001-WildBoy.gif``  
Topic is Dungeons & Dragons, ID is 001, name is "Wild Boy"

### Topics key:
| Number | Topic              |
|--------|--------------------|
| 0      | General            |
| 1      | Dungeons & Dragons |
| 2      | Film               |
| 3      | Music              |
| 4      | Reading            |
| 5      | Other              |
