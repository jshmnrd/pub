# Share Files

These are files that are meant to be shared with people. For example if I had a PDF I wanted site visitors to be able to download, I'd file it here.

## Naming Convention

File code system is derived from Johnny.Decimal.

First number is the topic,
the following three numbers are the ID,
Then a dash (-) and the name with no spaces.

Example: ``03001-ExplicitIcon.svg``  
Topic is music, ID is 001, name is "Explicit Icon"
